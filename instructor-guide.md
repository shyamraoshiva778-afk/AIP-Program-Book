# AIP Instructor Guide

> **For Teachers, Trainers, and Facilitators of the Academic Intelligence Program**

---

## 📋 Overview

This guide helps you deliver the AIP program effectively. Whether you are a school teacher, a private tutor, or a program coordinator, this document gives you everything you need to run the 15-day program.

---

## 🎯 Your Role as an Instructor

In AIP, you are NOT a traditional "lecturer." You are:

| Role | What You Do |
|------|-------------|
| **Guide** | Show students the path, then let them walk |
| **Facilitator** | Create the environment for learning to happen |
| **Coach** | Encourage, support, and push students to grow |
| **Demonstrator** | Show live examples of AI interactions |
| **Safety Monitor** | Ensure students use AI responsibly |

---

## 🏗️ Program Structure

```
Total Duration: 15 Days (can be stretched to 3-4 weeks)
Session Length: 45-60 minutes per day
Class Size: 10-30 students (ideal: 15-20)
Student Ages: 12-16 years (Classes 8-10)
```

### Daily Session Flow:

```
┌────────────────────────────────┐
│  WARM UP (5 min)               │
│  ├── Quick recap of yesterday  │
│  └── Today's objective         │
├────────────────────────────────┤
│  MAIN LESSON (20-25 min)       │
│  ├── Explain concept           │
│  ├── Show examples             │
│  └── Live AI demonstration     │
├────────────────────────────────┤
│  PRACTICAL ACTIVITY (15 min)   │
│  ├── Students try themselves   │
│  └── You walk around & help    │
├────────────────────────────────┤
│  SHARE & DISCUSS (10 min)      │
│  ├── Students share results    │
│  ├── Questions & answers       │
│  └── Key takeaways             │
├────────────────────────────────┤
│  WRAP UP (5 min)               │
│  ├── Summary                   │
│  ├── Homework assignment       │
│  └── Preview of next day       │
└────────────────────────────────┘
```

## 🧑‍🏫 Teaching Tips

### Before the Program:

1. **Test AI tools yourself** — Try every prompt in the book before class
2. **Prepare examples** — Have your own examples ready for each topic
3. **Check internet** — Ensure reliable WiFi for AI tool access
4. **Backup plan** — Have screenshots/recordings in case internet fails
5. **Know your students** — Understand their levels (Pass & Improve / High Scorers / Deep Learners)

### During Sessions:

1. **Go SLOW** — This is new for many students. Let them absorb
2. **Demonstrate LIVE** — Show the AI interaction on a projector/screen
3. **Encourage experimentation** — Let students try DIFFERENT prompts
4. **Use pair work** — Have students work in pairs, helping each other
5. **Celebrate wins** — When a student gets a great result, share it with the class
6. **Normalize mistakes** — Show that bad prompts = bad answers, and that's okay

### Handling Challenges:

| Challenge | Solution |
|-----------|----------|
| Student cannot access AI | Pair them with a student who can, or use your device |
| Internet is slow | Pre-load examples, use offline activities |
| Student uses AI to cheat | Have one-on-one conversation about ethics (Day 13) |
| Mixed ability levels | Use the 3-category system built into every chapter |
| Student finishes early | Challenge them with "Deep Learner" tasks |
| Student is struggling | Give them "Pass & Improve" tasks, build confidence |

## 📱 Live Demonstration Guide

For every chapter, here is what to demonstrate LIVE on screen:

| Day | Live Demo |
|-----|-----------|
| 1 | No AI needed — just discussion and inspiration |
| 2 | Open ChatGPT/DeepSeek, type greeting, show response |
| 3 | Show BAD prompt ("Tell me about science") vs GOOD prompt (RCT formula) |
| 4 | Upload a textbook page, run the Chapter Analysis prompt |
| 5 | Generate flashcards from a chapter in real-time |
| 6 | Upload 2-3 previous year papers, show analysis |
| 7 | Create a study plan with AI based on a student's input |
| 8 | Run a Socratic session live — let students answer AI's questions |
| 9 | Generate a concept map and mnemonics |
| 10 | Ask AI a tricky question, show how to verify it |
| 11 | Do a research deep dive on a student's chosen topic |
| 12 | Show Beginner vs Expert prompt results side-by-side |
| 13 | Read scenarios aloud, discuss as class |
| 14 | Build a learning plan with AI for a student |
| 15 | Show the certificate, have students write their letter |

## 📝 Assessment & Tracking

### Daily Check:

```
Student Name: __________  Date: Day ___

Participation:  □ Excellent  □ Good  □ Needs Improvement
Activity Completed:  □ Yes  □ Partial  □ No
Understanding:  □ Got it  □ Almost  □ Needs Help
Notes: ____________________________
```

### End-of-Program Assessment:

| Skill | How to Assess |
|-------|---------------|
| Prompt Writing | Give a topic, ask student to write a prompt |
| Chapter Analysis | Give a short chapter, ask for key points |
| Exam Intelligence | Give sample questions, ask to identify patterns |
| Memory Techniques | Ask student to explain spaced repetition |
| Critical Thinking | Give an AI answer, ask student to verify it |
| Ethics | Present a scenario, ask if it's ethical |

## 🗣️ Discussion Starters

Use these when conversation slows down:

- "What surprised you about what AI can do?"
- "What is ONE thing you will NEVER use AI for?"
- "How is studying with AI different from studying without it?"
- "What would you tell a friend who is curious about AIP?"
- "What is the most useful thing you have learned so far?"

---

## ⚠️ Important Reminders

```
🔴 NEVER share personal information with AI during demos
🔴 Always verify AI answers — model critical thinking
🔴 Do NOT assume all students have AI access — plan pairs
🔴 Be patient — some students will learn faster than others
🔴 Adapt the pace to your class — 15 days is a guide, not a rule
🔴 Most importantly: MAKE LEARNING FUN!
```
