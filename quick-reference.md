## 📱 The AI Chat Box — Your New Best Friend

Every AI tool has a **text box** at the bottom of the screen. This is where you type. Think of it like texting a super-smart friend.

```
┌─────────────────────────────────────────────────┐
│  AI Chat Window                                 │
├─────────────────────────────────────────────────┤
│                                                 │
│  AI: Hello! How can I help you today?           │
│                                                 │
│                                                 │
│  You: [____________________________]            │
│       [Send Button →]                           │
└─────────────────────────────────────────────────┘
```

**You type → AI responds → You type again → AI responds better**

---

## 🎯 The 5 Most Common Ways to Use the Chat Box

### 1. ASK a Question

Type your question directly and press Enter/Send.

```
────────────────────────────────────
📱 TYPE THIS: "What is photosynthesis?"
────────────────────────────────────
```

**AI responds with:** An explanation of photosynthesis.

**Then you can follow up:**
```
📱 TYPE THIS: "Make it simpler. I am in class 8."
────────────────────────────────────
```

---

### 2. GIVE a Role + Task (The RCT Formula)

```
────────────────────────────────────
📱 TYPE THIS: "Act as my Science tutor.
I am in class 9 and I don't understand
the Pythagorean theorem.
Explain it step by step with an example."
────────────────────────────────────
```

**Why this works:** You told AI WHO to be (tutor), YOUR situation (class 9, confused), and WHAT to do (explain step by step).

---

### 3. UPLOAD a File

Most AI tools have an **upload button** (📎 or + icon).

```
1. Click the upload button
2. Select your file (textbook page, notes, question paper)
3. Then type what you want AI to do with it:

📱 TYPE THIS: "I just uploaded a chapter on
the French Revolution. Please summarize it
in 5 key points with dates."
```

---

### 4. COPY-PASTE Text

If upload is not available, copy text from your book/notes and paste it.

```
📱 TYPE THIS: "Here is a paragraph from my textbook.
Please explain it in simple words:

[PASTE YOUR TEXT HERE]

What are the 3 most important things to remember?"
```

---

### 5. HAVE a Conversation (Follow-ups)

The magic happens when you keep talking.

```
YOU: "Explain gravity."
AI: [Answers]

YOU: "Give me a real-life example."
AI: [Example]

YOU: "I still don't get the part about mass.
Explain differently."
AI: [Different explanation]

YOU: "Now quiz me with 3 questions."
AI: [Quiz questions]
```

**Each follow-up makes the answer better for YOU.**

## 🎨 The Prompt Templates You Will Use Most

Here are the **top 10 prompts** you will use again and again. Save these!

### #1: Explain Anything
```
📱 TYPE THIS:
Act as my [Subject] tutor. I am in class [Class].
Explain [Topic] in simple words. Give an example
from daily life. Then check if I understood.
```

### #2: Summarize a Chapter
```
📱 TYPE THIS:
Summarize this chapter in 5-7 bullet points.
Focus on exam-important information.
Use simple language.
[PASTE/UPLOAD CHAPTER]
```

### #3: Create Flashcards
```
📱 TYPE THIS:
Create 15 flashcards from this chapter.
Format:
Q1: [Question]
A1: [Answer]
[PASTE/UPLOAD CHAPTER]
```

### #4: Quiz Me
```
📱 TYPE THIS:
Quiz me on [Topic]. Ask 10 questions.
Start easy, then get harder.
After each answer, tell me if I am correct.
If wrong, explain the correct answer.
```

### #5: Analyze Question Paper
```
📱 TYPE THIS:
Analyze this question paper and tell me:
1. Which topics have the most marks?
2. What types of questions appear most?
3. What patterns do you see?
[PASTE/UPLOAD QUESTION PAPER]
```

### #6: Create Study Plan
```
📱 TYPE THIS:
Create a [X]-day study plan for [Subject].
My exam is on [Date]. Chapters to cover: [List].
I can study [X] hours daily.
```

### #7: Explain Like I'm 10
```
📱 TYPE THIS:
Explain [Topic] like I am 10 years old.
Use very simple words. Use an analogy.
No complex terms without explanation.
```

### #8: Check My Answer
```
📱 TYPE THIS:
Here is my answer to a [marks] question on [Topic].
Please check it and tell me:
1. What did I get right?
2. What did I miss?
3. How can I improve it?

[YOUR ANSWER]
```

### #9: Compare Two Topics
```
📱 TYPE THIS:
Compare [Topic A] and [Topic B] in a table.
Show: Definition, Key Points, Examples,
and How They Are Connected.
```

### #10: Teach Me Step by Step
```
📱 TYPE THIS:
Teach me [Topic] step by step.
I am a complete beginner.
After each step, pause and ask if I understand.
Only move to the next step when I say "Ready."
```

## 🖊️ How to Get Better Answers (Quick Tips)

| If You Want... | Then Say... |
|----------------|-------------|
| Simpler answer | "Make it simpler. I am in class [X]." |
| More detail | "Give me more details about point 2." |
| An example | "Give me a real-life example." |
| A different explanation | "Explain it differently. I still don't understand." |
| Practice questions | "Now quiz me on this topic." |
| Shorter answer | "Summarize in 3 bullet points." |
| Check your work | "Check my answer and tell me what I missed." |
| Connect topics | "How does this connect to what I learned before?" |

## 🎬 What to Do When...

### ...AI gives a wrong answer
```
📱 TYPE THIS: "Are you sure? Double check that."
Then verify with your textbook.
```

### ...AI uses complex words
```
📱 TYPE THIS: "Use simpler words. I don't understand
these terms: [list terms]."
```

### ...AI gives too much information
```
📱 TYPE THIS: "Give me only the 3 most important points."
```

### ...You want to continue later
```
📱 TYPE THIS: "Summarize what we discussed so far.
I want to continue from here tomorrow."
```

### ...You want to save the conversation
Most AI tools have a "Copy" button or you can:
- Take a screenshot
- Copy and paste to a document
- Use the "Share" feature if available

## 🚫 What NOT to Do in the Chat Box

```
❌ Do NOT share personal information
   (Your full name, address, phone number, passwords)

❌ Do NOT ask AI to write your homework
   (Use AI to LEARN, not to copy)

❌ Do NOT trust everything AI says
   (Always verify with your textbook)

❌ Do NOT use AI during exams or tests
   (This is cheating — you will get caught)

❌ Do NOT give up if the first answer is bad
   (Refine your prompt — keep trying!)
```

## 📋 Chat Box Checklist

Before you type, ask yourself:

```
□  Did I tell AI who to be?     (Role)
□  Did I explain my situation?  (Context)
□  Did I say what I want?       (Task)
□  Did I give enough detail?    (Specifics)
□  Am I learning from this?     (Purpose)
```
